# CISP 401- Project 25- Java Date and Time Usage

## Overview
Java program that uses the java.time library.
Uses dates, times, formatting, and date calculations.

## What the Program Does

* Prints current date, time, and date-time
* Extracts date components like year, month, day, and weekday
* Creates custom dates and times
* Calculates time differences between dates
* Adds days to a date

## Key Ideas

* Java time API
* Date arithmetic and calculations
* Extracting calendar parts
